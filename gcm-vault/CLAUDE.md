## 我是誰

冠誠（品牌全名以 `00_brand/naming.md` 為準）是一家全漏斗成長顧問，核心不是「一站式行銷解決方案」，
而是三項具體、可獨立銷售的顧問服務：

1. **海外房地產企業顧問**
2. **逆向破解競業情報書**
3. **履歷重構與面試技巧**

三項服務彼此獨立計價、獨立交付，不得混為同一份報價或同一頁面。

## 硬規則（不可違反）

1. **品牌名稱**：對外一律使用「冠誠」，不得使用舊稱、英文縮寫混用，除非該檔案明確標註
   為法定登記全名（如合約抬頭）。
2. **三服務定價鐵律**：任何頁面、提案、合約引用的價格，只能來自 `01_services/` 中
   `status: active` 的檔案。看到任何其他地方（官網舊稿、舊聊天記錄、`99_archive/`）出現
   不同數字，一律視為過期，不得引用，須先到 `01_services/` 核對。
3. **格式規則**：
   - 中文文案不使用破折號（—、-- 皆不用），改用逗號或分句。
   - 條列項目結尾不加句號或逗號。
   - 服務命名以「系統」「制度」為核心詞（例如「90 天高信任成交系統」「每日競業雷達」），
     不使用「訂閱」「月費監控」這類會被誤解為單次消費的字眼——但合約與報價單上仍須
     如實揭露其訂閱制／週期性收費本質，對外行銷語言與合約法律語言分屬兩層，不可用行銷
     話術取代合約揭露義務。
4. **金流事實**：SurveyCake 是問卷與名單收集工具，不是金流工具；綠界（ECPay）才是實際
   收款的 API 型金流。任何文案不得暗示 SurveyCake 可以直接收款。零程式碼路線＝
   「SurveyCake 收資料 → 導向綠界收款連結頁 → SurveyCake 自訂感謝頁轉址完成通知 →
   自動回信」，不是「SurveyCake 一站完成金流」。
5. **合規紅線**：
   - 海外房地產顧問服務屬不動產經紀相關業務，內容須檢核是否觸及《不動產經紀業管理
     條例》規範的仲介／代銷行為邊界，顧問定位與經紀定位需明確切割，寫法以
     `06_facts/` 內查核紀錄為準。
   - 海外房產投報率／報酬率文案，必須註明數據來源與計算基礎，並附風險警語，不得使用
     保證獲利、穩賺等字眼。
   - SurveyCake 免費版每月回覆數上限 100 份，任何名單規模規劃超過此數字時，文案與
     SOP 須明確標註需升級付費方案或改用其他工具，不得假設免費版可無限承接。
6. **交付物標準化**：對客戶的正式交付一律以 PDF 呈現定稿版本（母版原始檔可為 Word／
   Excel，但交付客戶時輸出 PDF）。
7. **異步顧問溝通**：全文字化溝通（不依賴視訊會議）是本服務賣點，文案與 SOP 中一律
   正面表述，不得寫成「僅提供文字客服」這類貶義措辞。
